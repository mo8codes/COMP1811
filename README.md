
# Python Coursework 1

This project is a command-line tool for managing a family tree.

## Supported features

### printing entire family tree

When the user interface asks, choose option 1.

The family tree will be displayed in the following format:

```
Person
|
|- Mother
|    |
|    |- Mother
|    | ...
|    |
|    |- Father
|    ...
|
|- Father
     |
     | - Mother
     | ...
     |
     | - Father
     | ...
     
```
